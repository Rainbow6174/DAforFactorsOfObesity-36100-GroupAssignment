Dataset from https://archive.ics.uci.edu/dataset/544/estimation+of+obesity+levels+based+on+eating+habits+and+physical+condition

'stage_1_prep.ipynb'
Provides initial data checks, cleaning, and preparation before analysis.

'DSI_AT_STAGE1_EDA.ipynb'
Contains exploratory data analysis (EDA), including feature distributions, correlations, and key insights about the dataset.

'dsi_at_stage_2c_Modeling.ipynb'
Implements the modeling pipeline, including preprocessing, model training, evaluation, feature importance analysis, and result interpretation.

'Stage 1 Report'
Summarises the EDA findings and presents insights from the exploratory phase.

'Stage 2 Report'
Presents the modeling results, evaluation, analysis, and the final conclusions of the study.
